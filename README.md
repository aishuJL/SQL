# 🗄️ SQL Projects

This repository contains SQL projects built using MySQL — focused on real-world database design, structured querying, and relational data management.

---

##  What You'll Find Here

Each project in this repository is centered around a real-world use case and covers core SQL concepts including:

- Relational database design with multiple tables
- Primary and foreign key constraints
- Inner joins across related tables
- Aggregate functions and grouped reporting
- Date-based filtering and formatting
- Real-world data like users, records, transactions, and logs

---

##  Tech Stack

- **Database** — MySQL 8.0
- **Tool** — MySQL Workbench
- **Editor** — Cursor (VS Code)
- **Version Control** — Git & GitHub

---

##  About This Repo

All projects here are hands-on implementations built from scratch — from database creation to inserting data and writing meaningful queries. Each project has its own folder with organized SQL files and documentation.
